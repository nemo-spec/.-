Platforms

Frontend

Built with modern web technologies and hosted on Vercel . The frontend delivers a fast, responsive, and polished user experience with excellent performance and global edge caching. Vercel enables seamless deployments from GitHub with automatic previews and instant rollbacks.

Authentication

User authentication and authorization are handled securely using Clerk (primary) with AWS Cognito as an alternative option. This setup provides seamless social login, email/password authentication, user management, and session handling while maintaining high security standards and excellent developer experience.

Backend

The server-side logic and APIs are powered by Railway . This platform provides a reliable, scalable environment for running the application backend with easy environment management and database integration. Railway allows for quick deployments and excellent observability during development and production.

Storage

File and media storage is handled through Supabase Storage (primary) with Railway volumes as backup. This setup supports secure uploading, serving, and management of user-generated content such as profile pictures, event images, and documents with proper access control.

Databases

Data is managed with Supabase (primary) and Railway as a secondary option. Supabase offers a powerful PostgreSQL database with real-time capabilities, authentication helpers, and storage integration. This combination ensures data reliability, scalability, and developer-friendly querying.

Version Control & Deployment

The entire codebase is managed using GitHub. It serves as the central hub for version control, collaboration, continuous integration, and automated deployments to Vercel and Railway. GitHub enables clean workflows with pull requests, issue tracking, and project documentation.
